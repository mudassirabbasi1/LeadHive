# LeadHive Landing Page

A responsive landing page for a lead-generation and outreach service. The page presents the offer, explains the workflow, lists service tiers, and includes a contact/order form.

## What It Includes

- Sticky navigation.
- Hero section with primary call to action.
- Business niche examples.
- Step-by-step service explanation.
- Feature and pricing sections.
- Contact/order form.
- Responsive CSS for mobile and desktop layouts.

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## View Locally

```bash
git clone https://github.com/mudassirabbasi1/LeadHive.git
cd LeadHive
```

Open `index.html` in your browser.

## Notes

This is the static marketing front end. The more complete full-stack automation system is available in [leadhive-automation](https://github.com/mudassirabbasi1/leadhive-automation).
